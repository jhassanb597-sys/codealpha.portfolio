Building Your First JavaScript Calculator: A Step-by-Step Guide

Apr 25, 2026 · 6 min read · by Jahangir

Introduction

Creating a calculator is a rite of passage for every front-end developer. It's the perfect project to practice DOM manipulation, event handling, and logic flow.

In this guide, we will build a fully functional, keyboard-supported calculator using HTML5, CSS Grid, and Vanilla JavaScript.

1. The HTML Structure

We need a display screen and a grid of buttons. We will group buttons in a div container for easy styling.

2. Styling with CSS Grid

CSS Grid is the ideal tool for a calculator. It allows us to create a perfectly aligned keypad with minimal effort.

.buttons {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: 10px;
  padding: 20px;
}

#display {
  width: 100%;
  height: 50px;
  text-align: right;
  font-size: 2rem;
  padding: 10px;
}

.btn {
  padding: 20px;
  font-size: 1.5rem;
  cursor: pointer;
}

3. The JavaScript Logic

The logic involves three parts:

  • Capturing clicks.
  • Updating the display.
  • Calculating the result using the built-in eval() function (or a safer math parser for production).
const display = document.getElementById('display');
const buttons = document.querySelectorAll('.btn');

buttons.forEach(button => {
  button.addEventListener('click', () => {
    const value = button.dataset.value;
    
    if (value === '=') {
      display.value = eval(display.value); // Basic calculation
    } else if (value === 'C') {
      display.value = '';
    } else {
      display.value += value;
    }
  });
});

4. Adding Keyboard Support

A great calculator should respond to keyboard inputs. We listen for the keydown event on the document.

document.addEventListener('keydown', (event) => {
  const key = event.key;
  
  // Allow numbers and specific operators
  if (/[0-9/*\-+]/.test(key)) {
    display.value += key;
  }
  if (key === 'Enter') {
    display.value = eval(display.value);
  }
  if (key === 'Escape') {
    display.value = '';
  }
});

Best Practices for Your Calculator

Security Note

While eval() is fine for a learning project, avoid it in production environments. Use a math library like math.js to parse expressions safely.
